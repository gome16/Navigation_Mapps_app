# Navigation_Mapps

## サイト概要
### サイトテーマ
指定の位置情報とその場所のトピックを投稿でき、その日起こったことを共有できるSNS
​
### テーマを選んだ理由
ニュースや新聞などで聞きなれない地名などの説明を聞いても国や地域の場所がわからないため、あまり理解が進まないということに悩みを抱えておりました。
実際「A国がB国に攻撃を仕掛けた」という文面だけではあまり理解できる人は少ないと感じております。
また「C川が氾濫したので避難してください」などの具体的にどの部分が氾濫しているのかなど説明する側も受け取る側も声明や文面では難しいと思いました。
これら諸問題を解決するため地図上にピンや範囲を投稿することにより、具体的な地理が明確になり説明や理解の助けになるのではないかと考え、このようなポートフォリオ作成に至りました。
​
### ターゲットユーザ
ニュースなどの起きた場所に関する情報を発信したい人またはそれを受け取りたい人
投資情報として地政学分析を発信したりそれを参考にする人
避難情報などを発信する自治体やそれを必要とする人
​

### 主な利用シーン
ニュースや新聞等を見てもその地名がわからなかったり、地図が載っていても縮尺が大きすぎて具体的な場所がわからない時
その国に投資を考えようとした際に地政学リスクなど手早くその国の情報が見たい時
災害情報を素早く発信しなければならない、あるいはそれを確認しなければならない時
